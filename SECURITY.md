# Security Policy

## Supported Versions

Security fixes are applied to the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report privately with:

- Vulnerability summary
- Affected files/components
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

Until a dedicated private contact is configured, open a private advisory/security report in GitHub for this repository.

## Response Expectations

- Initial triage target: within 7 days
- Remediation target: depends on severity and complexity
- Public disclosure: after a fix is available or a coordinated disclosure timeline is agreed

## Scope Notes

This app downloads and executes local model inference artifacts. Treat model URLs and files as untrusted input and validate source integrity when possible.
